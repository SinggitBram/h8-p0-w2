Soal 2:

Algoritma:
1. Tampung number apapun ke variabel tahun
2. Hitung apakah tahun habis dibagi 4 dan tidak habis dibagi 100, apabila true lanjut ke step 4
3. Hitung apakah tahun habis dibagi 4 dan habis dibagi 100 dan habis dibagi 400 apabila true lanjut ke step 4
4. Tampilkan "Tahun Kabisat"
5. Apabila hasil hitungan false tampilkan "Bukan Tahun Kabisat"


Pseudocode:

STORE "tahun" with any value 
IF "tahun" mod 4 equals to 0 AND "tahun" mod 100
	not equals to 0
	DISPLAY "Tahun kabisat"
END IF

ELSE IF "tahun" mod 4 equals to 0 AND "tahun" mod 100
	equals to 0 AND "tahun" mod 400 equals to 0 
	DISPLAY "Tahun kabisat"
ELSE
	DISPLAY "Bukan tahun kabisat"
END IF

Soal 4:

1.tampung nilai 0 ke variabel murid

4.apabila kuku panjang bernilai false maka tampilkan "dipuji"


STORE "murid" to 1

  	DETERMINE "kuku panjang" as True or False
